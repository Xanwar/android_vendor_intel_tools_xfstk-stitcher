xfstk-stitcher CLI Usage

xfstk-stitcher Help Ouput

Type xfstk-stitcher with no additional parameters and you will see the help message from the prompt.

    Options              Description

   -h --help             print this help message and exit
   -k platform.xml       specify the platform XML file
   -c Config.txt         specify the Configuration file
   -G                    Generate final IFWI image from signed components
   -M                    Generate unsigned   MIP file
   -A                    Generate unsigned Upper Firmware (IA, Supplemental, Hooks)
   -J                    Generate Unsigned Soft Fuse File
   -V --version          print the version number and exit.

xfstk-stitcher Basic Usage

Use the following command line to generate final IFWI image from signed components

        xfstk-stitcher -k platform.xml -c config.txt -G

Use the following command line to generate Unsigned Upper Firmware (IA, Supplemental, Hooks, FIP Header)

      xfstk-stitcher -k platform.xml -c config.txt -A

Use the following command line to generate Unsigned MIP (UMIP & SMIP)

        xfstk-stitcher -k platform.xml -c config.txt -M

Generate Unsigned Soft Fuse file:

        xfstk-stitcher -k platform.xml -c config.txt -J
xFSTK Consumes Hashes: 
        PlatformType = CLVPB0
        ImageType = FWUSB

        'to Generate Unsigned Soft Fuse file "xfstk-stitcher -k platform.xml -c config.txt -J"

        'Key used to sign Soft fuse file
        SoftFuse PublicKey = ./C0_234_public.key

        '----------- Public Key Hash Files -------------------
        Public KeyHash2 = ./publicKeyHash2.bin
        Public KeyHash3 = ./publicKeyHash3.bin
        Public KeyHash4 = ./publicKeyHash4.bin

        XML = ./XML_PR_F4.22_DEV.xml
        ImageName = ./SoftFuse.bin
xFSTK Creates Hashes: 
        PlatformType = CLVPB0
        ImageType = FWUSB

        'to Generate Unsigned Soft Fuse file "xfstk-stitcher -k platform.xml -c config.txt -J"

        'Key used to sign Soft fuse file
        SoftFuse PublicKey = ./C0_234_public.key

        'Key Hashes to be programmed
        Public Key2 = ./C0_234_public.key
        Public Key3 = ./C0_234_public.key
        Public Key4 = ./C0_234_public.key

        XML = ./XML_PR_F4.22_DEV.xml
        ImageName = ./SoftFuse.bin

Use the following command line to generate OS Images

        xfstk-stitcher -k platform.xml -c config.txt

Reference Configuration Files

Configuration files are installed into:

  • /usr/share/xfstk-stitcher on linux platforms
  • C:\Program Files (x86)\xFSTK\stitcher on Windows(R) 7 64-bit platforms
  • C:\Program Files\xFSTK\stitcher on Windows(R) 32-bit platforms.

This documentation provides additional example configuration files.
