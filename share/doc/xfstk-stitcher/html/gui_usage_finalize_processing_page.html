Step 4: Finalize Processing

Input Validation

The Finalize Processing page performs various checks on the inputs and outputs of a given target. Validation errors are marked with an 'X' and details which describe the issues are displayed in the table.

Run

When all ingredients are valid, the [Run] button will become enabled. Clicking the run button starts the stitching process. The Output window will display the:

  1. Command line used
  2. Location of the XML/Configuration files
  3. Location of the image
  4. Elapsed Stitching Time
