Step 3: Customize Output

Clovertrail Plus GP Pin Programming Header

The GP Pin Programming Header contains customizable information for programming the following:

  • No Connect Indication
  • Open Drain Enable
  • Direction
  • Stand-by Level
  • Wake Make or Hold Output State
  • Pull Direction
  • Pull Strength
GP Pin Programming Header


Clovertrail Plus GPIO Configuration Registers

The GPIO Configuration Registers contain customizable information for programming the following:

  • Alt Function [GAFR]
  • Direction [GPDR]
  • Level [GPSR]
  • Rise [GRER]
  • Fall [GFER]
