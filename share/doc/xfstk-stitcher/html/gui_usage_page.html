Introduction

The xFSTK Stitcher Lite application provides a graphical user interface with stitching capabilities to generate an IFWI, Unsigned MIP, Unsigned IA, and single or multi OS image binary. The Start Page describes the workflow to stitch a given target. The steps include:

  1. Select Target: Choose SOC device and output formats (if applicable)
  2. Load Ingredients: Browse for the binaries to stitch
  3. Customize Output: Modify header values (if applicable)
  4. Final Processing: Generate the xml file, configuration file, and binary output
